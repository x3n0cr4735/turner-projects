# NEON GENESIS

**Engineering Portfolio by Turner Wagner, Age 11**

A collection of original vehicle and system designs spanning ground, air, space, and underwater domains.

## Projects

### ⬡ Magnetic Elevator (ME-2025)
A maglev elevator system using opposing magnetic poles for frictionless vertical transport. Features computer-controlled guidance and eliminates traditional cables and mechanical wear.

### △ T-47/M Supersonic Transport
An improvement on NASA's oblique wing concept with dual swept wings for enhanced stability. Wide-body design for passenger capacity with supersonic cruise capability.

### ◇ E-109 Spaceplane
A two-stage hybrid spaceplane with retractable wings for atmospheric flight and a detachable rocket stage for orbital insertion.

### ◎ AV-14 "Toxic" Deep-Sea Miner
A manned submersible for deep-sea mining operations featuring robotic extraction arms, mineral transport tubes, and terrain-adaptive legs.
